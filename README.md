Requirements
------------

This front-end app requires the following to run:

  * [Node.js][node] 0.10+
  * [npm][npm] (normally comes with Node.js)


Usage
-----
When is installed  [npm][npm]:

```sh
npm i 
```

```sh
npm start 
```
